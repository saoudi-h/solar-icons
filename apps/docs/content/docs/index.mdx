**A comprehensive icon library for modern applications.** Solar provides 1,246 unique icons, each available in six distinct styles, designed to be modular, flexible, and easy to integrate into your projects.

With a total of 7,476 icon variations, Solar is built to meet the needs of today's web and mobile applications.

## Why Choose Solar?

- **Extensive Collection**: 1,246 unique icons across more than 30 categories.
- **Multi-Style Support**: Each icon is available in six styles: `Bold`, `Linear`, `Outline`, `BoldDuotone`, `LineDuotone`, and `Broken`.
- **Performance-Focused**: Choose between a versatile, all-in-one package or a highly optimized, lightweight package for performance-critical applications.
- **React Ready**: First-class support for React, with features like global configuration and server-side rendering.
- **Vue & Nuxt Ready**: Full-featured Vue components with global providers and Nuxt module with auto-import.
- **Modern Frameworks**: Support for Svelte 5, SolidJS, and Angular 17-21 with modern patterns.
- **Developer-Friendly**: A well-structured and maintainable codebase makes the library easy to use and extend.
