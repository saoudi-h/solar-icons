## Features

- **7,476 variations**: 1,246 unique icons across 30+ categories, each in six styles (`Bold`, `Linear`, `Outline`, `BoldDuotone`, `LineDuotone`, and `Broken`).
- **Optimization choices**: Choose standard packages for dynamic style switching, or performance packages (`-perf`) for minimal bundle sizes.
- **Framework support**: Native packages for React, React Native, Vue, Nuxt, Svelte 5, SolidJS, and Angular.
- **SSR & Next.js ready**: Works in Server-Side Rendering (SSR) and React Server Components (RSC) environments.
