Solar Icons organizes its 1,246 icons across categories like Arrows, Devices, Files, and Users. Each icon belongs to one category.

## Browse

Use the [Icons page](/icons) to search and filter icons by category with live style switching, color controls, and duotone configuration.

## Import by category

Categories are not importable as a group in v3. Import icons individually for the best tree-shaking.

```jsx
import { HeartIcon } from '@solar-icons/react/bold/heart'
import { SettingsIcon } from '@solar-icons/react/outline/settings'
```
