Install the package for your framework. Every framework gets one package. No separate performance or reactive variant.

## React

Requires React 16.8+.

```package-install
@solar-icons/react
```

## Vue

Requires Vue 3.0+.

```package-install
@solar-icons/vue
```

## Nuxt

Nuxt 3+ module with auto-import and configuration.

```shell
npx nuxi module add @solar-icons/nuxt
```

Add the module to `nuxt.config.ts`:

```ts
export default defineNuxtConfig({
    modules: ['@solar-icons/nuxt'],
    solarIcons: {
        namePrefix: 'Solar',
        autoImport: true,
    },
})
```

## Svelte

Svelte 5 components using runes. Requires Svelte 5.0+.

```package-install
@solar-icons/svelte
```

## SolidJS

Requires SolidJS 1.6+.

```package-install
@solar-icons/solid
```

## Angular

Requires Angular 17+.

```package-install
@solar-icons/angular
```

## React Native

Requires React 16.8+, React Native 0.72+, and `react-native-svg` 13.0+.

```package-install
@solar-icons/react-native react-native-svg
```
