v3 unifies the library under a single API across all frameworks. This guide covers every breaking change.

## Package renames

| v1/v2 Package                 | v3 Package                  | Notes                                                            |
| ----------------------------- | --------------------------- | ---------------------------------------------------------------- |
| `@solar-icons/react`          | —                           | Removed. Use `@solar-icons/react/dynamic` for runtime switching. |
| `@solar-icons/react-perf`     | `@solar-icons/react`        | Classic mode is now the default.                                 |
| `@solar-icons/react-reactive` | —                           | Removed. Dynamic switching merged into `react`.                  |
| `@solar-icons/vue`            | `@solar-icons/vue`          | Same name. API changed.                                          |
| `@solar-icons/vue-reactive`   | —                           | Removed. Dynamic switching merged into `vue`.                    |
| `@solar-icons/svelte`         | `@solar-icons/svelte`       | Same name. Updated to Svelte 5 runes.                            |
| `@solar-icons/solid`          | `@solar-icons/solid`        | Same name.                                                       |
| `@solar-icons/angular`        | `@solar-icons/angular`      | Same name. Signals + standalone.                                 |
| `@solar-icons/react-native`   | `@solar-icons/react-native` | Same name.                                                       |
| `@solar-icons/nuxt`           | `@solar-icons/nuxt`         | Same name. Wraps new Vue package.                                |

## Import paths

**Before:**

```jsx
import { Home } from '@solar-icons/react/Bold'
```

**After:**

```jsx
import { HomeIcon } from '@solar-icons/react/bold'
```

Kebab-case style directories (`bold`, `bold-duotone`, `broken`, `line-duotone`, `linear`, `outline`).

## Component naming

The component name drops the style suffix when the style is in the import path.

**Before:**

```jsx
import { Home } from '@solar-icons/react'
;<Home weight="Bold" />
```

**After (per-style):**

```jsx
import { HomeIcon } from '@solar-icons/react/bold'
;<HomeIcon />
// Name is just HomeIcon — style is in the path
```

**After (top-level, all styles):**

```jsx
import { HomeBoldIcon } from '@solar-icons/react'
;<HomeBoldIcon />
// Name includes style suffix — path doesn't specify style
```

## Category imports removed

**Before:**

```jsx
import { Arrows } from '@solar-icons/react/category'
;<Arrows.ArrowUp weight="Bold" />
```

**After:**

```jsx
import { ArrowUpIcon } from '@solar-icons/react/bold/arrow-up'
;<ArrowUpIcon />
```

## `weight` prop removed from per-style icons

**Before:**

```jsx
import { Heart } from '@solar-icons/react-perf'
;<Heart weight="Bold" />
```

**After (static):**

```jsx
import { HeartIcon } from '@solar-icons/react/bold/heart'
;<HeartIcon />
```

**After (dynamic):**

```jsx
import { HeartIcon } from '@solar-icons/react/dynamic'
;<HeartIcon weight="Bold" />
```

## `mirrored` prop removed

**Before:**

```jsx
<ArrowRight mirrored />
```

**After:**

```jsx
import { ArrowRightIcon } from '@solar-icons/react/bold'
;<ArrowRightIcon style={{ transform: 'scaleX(-1)' }} />
```

## SSR subpath removed

**Before:**

```jsx
import { Home } from '@solar-icons/react/ssr'
```

**After:**

```jsx
import { HomeIcon } from '@solar-icons/react/bold'
```

## Vue: no more vue-reactive

**Before:**

```jsx
import { Heart } from '@solar-icons/vue-reactive'
```

**After:**

```vue
<script setup>
import { HeartIcon } from '@solar-icons/vue/dynamic'
</script>
<template>
    <HeartIcon weight="Bold" />
</template>
```

## New: SolarProvider

```jsx
import { SolarProvider } from '@solar-icons/react'

function App() {
    return (
        <SolarProvider color="#3b82f6" size={24} strokeWidth={1.5}>
            <YourApp />
        </SolarProvider>
    )
}
```

## New: Duotone

```jsx
import { HeartIcon } from '@solar-icons/react/bold-duotone'

;<HeartIcon color="#3b82f6" secondaryColor="#f59e0b" secondaryOpacity={0.4} />
```

## New: Stroke width

```jsx
import { SettingsIcon } from '@solar-icons/react/linear'

;<SettingsIcon strokeWidth={2} size={24} />
```

## Quick checklist

1. Replace `@solar-icons/react-perf` with `@solar-icons/react`
2. Convert PascalCase import directories to kebab-case
3. Remove `weight` prop or switch to `dynamic` imports
4. Remove `mirrored` prop (use CSS `scaleX(-1)`)
5. Remove `/ssr` and `/category` from import paths
6. Add style suffix to names when importing from top-level
7. Use `SolarProvider` for global defaults instead of per-icon props
