Solar Icons gives you 1,246 distinct icons in six styles: **Bold**, **Linear**, **Outline**, **BoldDuotone**, **LineDuotone**, and **Broken**. With 7,476 total variations and packages for React, Vue, Svelte, Solid, Angular, and React Native.

## What changed in v3

The v3 release unifies the library under a single API across every framework.

- **One package per framework.** `@solar-icons/react`, `@solar-icons/vue`, `@solar-icons/svelte`, `@solar-icons/solid`, `@solar-icons/angular`, `@solar-icons/react-native`, and `@solar-icons/nuxt`. No more `-perf` and `-reactive` variants.
- **Classic mode by default.** Every icon ships with a single style per component, tree-shakable at the import level. Dynamic style switching uses a separate `dynamic` import path.
- **CSS variable cascade.** A `SolarProvider` at the root sets `--solar-color`, `--solar-size`, `--solar-stroke-width`, and duotone variables. Icons inherit these without re-rendering.
- **Duotone built in.** `secondaryColor` and `secondaryOpacity` props control the duotone accent path. No wrapper components or hacky CSS overrides.
- **Stroke width control.** Linear, Broken, and LineDuotone icons accept `strokeWidth`. Defaults to `1.5`.
- **Icon suffix on component names.** `HomeBoldIcon`, not `Home`. Makes multi-style imports unambiguous.
- **Kebab-case import paths.** `@solar-icons/react/bold/heart`, not `@solar-icons/react/Bold/Heart`. Matches ecosystem conventions.
