Pick your starting point:

## From `@solar-icons/react-perf`

`react-perf` was the classic package — one style per component, tree-shakable, no dynamic switching. In v3, this model becomes the default.

### Package

```bash
npm uninstall @solar-icons/react-perf
npm install @solar-icons/react
```

### Component naming

Add `Icon` suffix. The style suffix depends on the import path:

**Before (top-level):**

```jsx
import { HomeBold, SettingsLinear } from '@solar-icons/react-perf'
;<HomeBold color="green" size={24} />
```

**After (top-level):**

```jsx
import { HomeBoldIcon } from '@solar-icons/react'
;<HomeBoldIcon color="green" size={24} />
```

**Before (per-style):**

```jsx
import { Home } from '@solar-icons/react-perf/bold'
;<Home size={24} />
```

**After (per-style):**

```jsx
import { HomeIcon } from '@solar-icons/react/bold'
;<HomeIcon size={24} />
```

### Import paths

Style directories use kebab-case: `Bold` → `bold`, `BoldDuotone` → `bold-duotone`.

### `mirrored` removed

```jsx
// Before
<ArrowRight mirrored />

// After
<ArrowRightIcon style={{ transform: 'scaleX(-1)' }} />
```

### `size` default changed

Default was `1em`, now `24px`. Explicit sizes work as before.

### New features

`SolarProvider`, `secondaryColor`/`secondaryOpacity` for duotone, and `strokeWidth` are now available. See the [React package page](/docs/v3/packages/react).

## From `@solar-icons/react` (dynamic)

The old `@solar-icons/react` was the dynamic package — all six styles per component, runtime switching via `weight`. In v3, this is accessed from the `dynamic` import path.

### Package

Same package, different import path:

```jsx
// Before
import { Heart } from '@solar-icons/react'
;<Heart weight="Bold" />
```

```jsx
// After
import { HeartIcon } from '@solar-icons/react/dynamic'
;<HeartIcon weight="Bold" />
```

### Component naming

Add `Icon` suffix: `Home` → `HomeIcon`.

### Category imports removed

**Before:**

```jsx
import { Arrows } from '@solar-icons/react/category'
;<Arrows.ArrowUp weight="Bold" />
```

**After:**

```jsx
import { ArrowUpIcon } from '@solar-icons/react/dynamic/arrow-up'
;<ArrowUpIcon weight="Bold" />
```

### SSR subpath removed

**Before:**

```jsx
import { Home } from '@solar-icons/react/ssr'
```

**After:**

```jsx
import { HomeIcon } from '@solar-icons/react/bold'
```

### New features

`SolarProvider`, `secondaryColor`/`secondaryOpacity`, and `strokeWidth` are now available. See the [React package page](/docs/v3/packages/react).
