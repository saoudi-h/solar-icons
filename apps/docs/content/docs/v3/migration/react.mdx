## Package

`@solar-icons/react-perf` becomes `@solar-icons/react`. The old `@solar-icons/react` (dynamic-switching) is now accessed via `@solar-icons/react/dynamic`.

```bash
npm uninstall @solar-icons/react-perf
npm install @solar-icons/react
```

## Import paths

Kebab-case style directories:

**Before:**

```jsx
import { Home } from '@solar-icons/react/Bold'
```

**After (per-style):**

```jsx
import { HomeIcon } from '@solar-icons/react/bold'
```

**After (top-level, all styles):**

```jsx
import { HomeBoldIcon } from '@solar-icons/react'
```

## Component naming

When the style is in the import path, the name drops the style suffix:

| Before       | After                                                            |
| ------------ | ---------------------------------------------------------------- |
| `Home`       | `HomeIcon` (per-style) or `HomeBoldIcon` (top-level)             |
| `ArrowRight` | `ArrowRightIcon` (per-style) or `ArrowRightBoldIcon` (top-level) |

## `weight` prop removed

Use dynamic import for runtime switching:

**Before:**

```jsx
import { Heart } from '@solar-icons/react'
;<Heart weight="Bold" />
```

**After:**

```jsx
import { HeartIcon } from '@solar-icons/react/dynamic'
;<HeartIcon weight="Bold" />
```

## Category imports removed

**Before:**

```jsx
import { Arrows } from '@solar-icons/react/category'
;<Arrows.ArrowUp weight="Bold" />
```

**After:**

```jsx
import { ArrowUpIcon } from '@solar-icons/react/bold/arrow-up'
;<ArrowUpIcon />
```

## `mirrored` prop removed

**Before:**

```jsx
<ArrowRight mirrored />
```

**After:**

```jsx
<ArrowRightIcon style={{ transform: 'scaleX(-1)' }} />
```

## SSR subpath removed

**Before:**

```jsx
import { Home } from '@solar-icons/react/ssr'
```

**After:**

```jsx
import { HomeIcon } from '@solar-icons/react/bold'
```

## New: SolarProvider

```jsx
import { SolarProvider } from '@solar-icons/react'

;<SolarProvider color="#3b82f6" size={24} strokeWidth={1.5}>
    <App />
</SolarProvider>
```

## New: Duotone and stroke width

```jsx
<HeartIcon secondaryColor="#f59e0b" secondaryOpacity={0.4} />
<SettingsIcon strokeWidth={2} />
```
